    1. 如何在Windows下使用 makefile命令
      1. 下载make工具
      2. MinGW环境下进行编译
      3. 添加到Path中
      4. 检测是否安装好

    如何在Windows下使用 makefile命令

    makefile工具

    下载make工具

    官网

    Make - GNU Project - Free Software Foundation

    点击下载

    选择一个tar.lz或者tar.gz下载

    下载完解压到本地

    MinGW环境下进行编译

    【MinGW】为例

    powershell输入

    build_w32.bat gcc

    获取到编译好的二进制文件，文件名位gnumake.exe 在GccRel文件夹下

    添加到Path中

    以后在终端中输入 gunmake便可以使用了

    想输入make使用，可以复制一份gunmake.exe改名位make.exe即可享用

    检测是否安装好

    make -v
gunmake -v
