    в–  putвҖ”вҖ”еҸ‘йҖҒдёҖдёӘ PUT иҜ·жұӮпјҢеҗ‘жҢҮе®ҡиө„жәҗдёҠдј жңҖж–°еҶ…е®№гҖӮ

    в–  patchвҖ”вҖ”еҸ‘йҖҒдёҖдёӘ PATCH иҜ·жұӮпјҢеҗҢ PUT зұ»дјјпјҢеҸҜд»Ҙз”ЁдәҺйғЁеҲҶеҶ…е®№жӣҙж–°гҖӮ

    в–  deleteвҖ”вҖ”еҸ‘йҖҒдёҖдёӘ DELETE иҜ·жұӮпјҢеҗ‘жҢҮе®ҡиө„жәҗеҸ‘йҖҒдёҖдёӘеҲ йҷӨиҜ·жұӮгҖӮ

    еҸҜд»ҘзңӢеҲ°пјҢrequests дҪҝз”ЁдёҺ HTTP иҜ·жұӮж–№жі•еҗҢеҗҚзҡ„ API жқҘжҸҗдҫӣзӣёеә”зҡ„ HTTP иҜ·жұӮжңҚеҠЎпјҢд»ҺиҖҢйҷҚдҪҺдәҶзј–зЁӢдәәе‘ҳзҡ„еӯҰд№ е’Ңи®°еҝҶжҲҗжң¬гҖӮеҸҰеӨ–пјҢиҝҷдәӣ API ж–№жі•йғҪи°ғз”ЁеҗҢдёҖдёӘеҹәзЎҖж–№жі•пјҢеӣ жӯӨеңЁи°ғз”ЁеҸӮж•°зҡ„дҪҝз”ЁдёҠд№ҹеҹәжң¬дҝқжҢҒдёҖиҮҙгҖӮ
