    WSL2&Hyper-V

    жң¬дәәеңЁWindowsдёҠдёӢиҪҪ Docker Desktop for Windowsж—¶пјҢйҒҮеҲ°дәҶиҰҒејҖеҗҜ Hyper-V е’Ң WSL2

    жң¬дәәдә§з”ҹдәҶжө“еҺҡзҡ„е…ҙи¶ЈпјҢе№¶дё”жғіе°қиҜ•з ”з©¶еӯҰд№ дёҖдёӢ

    WSL2

    1.1 д»Җд№ҲжҳҜ WSL2

    е®ҳж–№д»Ӣз»Қ:

    Windows Subsystem for Linux Documentation | Microsoft Learn

    WSL е…Ёз§° : Windows Subsystem for Linux Documentation

    жҰӮиҝ°пјҡWSL жҳҜеҫ®иҪҜеңЁ Windows 10 дёӯеј•е…Ҙзҡ„дёҖдёӘе…је®№еұӮпјҢе…Ғи®ёз”ЁжҲ·еңЁдёҚйңҖиҰҒиҷҡжӢҹжңәжҲ–еҸҢеҗҜеҠЁзҡ„жғ…еҶөдёӢиҝҗиЎҢ Linux дәҢиҝӣеҲ¶еҸҜжү§иЎҢж–Үд»¶пјҲELF ж јејҸпјүгҖӮ

    жһ¶жһ„пјҡWSL йҖҡиҝҮе°Ҷ Linux зҡ„зі»з»ҹи°ғз”Ёзҝ»иҜ‘жҲҗ Windows еҶ…ж ёзҡ„зі»з»ҹи°ғз”ЁжқҘе·ҘдҪңпјҢеӣ жӯӨ Linux зЁӢеәҸиғҪеӨҹеңЁ Windows дёҠзӣҙжҺҘиҝҗиЎҢгҖӮWSL дёҚжҳҜзңҹжӯЈзҡ„ Linux еҶ…ж ёпјҢиҖҢжҳҜдёҖдёӘе…је®№еұӮгҖӮ

    жҖ§иғҪпјҡз”ұдәҺжІЎжңүе®Ңж•ҙзҡ„ Linux еҶ…ж ёпјҢWSL зҡ„жҖ§иғҪеңЁжҹҗдәӣ I/O еҜҶйӣҶеһӢд»»еҠЎдёӯеҸҜиғҪдјҡеҸ—еҲ°йҷҗеҲ¶пјҢдҪҶеҜ№дәҺеӨ§еӨҡж•°ејҖеҸ‘д»»еҠЎе·Із»Ҹи¶іеӨҹгҖӮ

    WSL2

    жҰӮиҝ°пјҡWSL2 жҳҜ WSL зҡ„еҚҮзә§зүҲжң¬пјҢж—ЁеңЁи§ЈеҶі WSL зҡ„дёҖдәӣжҖ§иғҪе’Ңе…је®№жҖ§й—®йўҳгҖӮ

    жһ¶жһ„пјҡWSL2 еј•е…ҘдәҶдёҖдёӘе®Ңж•ҙзҡ„ Linux еҶ…ж ёпјҢиҝҗиЎҢеңЁдёҖдёӘиҪ»йҮҸзә§зҡ„иҷҡжӢҹжңәпјҲHyper-VпјүдёӯгҖӮиҝҷж„Ҹе‘ізқҖ WSL2 жҸҗдҫӣдәҶдёҺзңҹе®һ Linux зҺҜеўғжӣҙй«ҳзҡ„е…је®№жҖ§гҖӮ

    жҖ§иғҪпјҡз”ұдәҺдҪҝз”ЁдәҶзңҹжӯЈзҡ„ Linux еҶ…ж ёпјҢWSL2 зҡ„ I/O жҖ§иғҪе’Ңж•ҙдҪ“е…је®№жҖ§жҳҫи‘—жҸҗеҚҮпјҢе°Өе…¶жҳҜеңЁеӨ„зҗҶж–Үд»¶зі»з»ҹе’ҢзҪ‘з»ңж“ҚдҪңж—¶гҖӮWSL2 зҡ„ж–Үд»¶и®ҝй—®йҖҹеәҰжҜ” WSL еҝ«еҫ—еӨҡпјҢе°Өе…¶жҳҜеҜ№еӨ§йҮҸе°Ҹж–Үд»¶зҡ„ж“ҚдҪңгҖӮ

    еӯҳеӮЁпјҡWSL2 дҪҝз”ЁиҷҡжӢҹзЎ¬зӣҳпјҲVHDпјүжқҘеӯҳеӮЁж–Үд»¶пјҢиҖҢдёҚжҳҜзӣҙжҺҘеңЁ Windows ж–Үд»¶зі»з»ҹдёҠж“ҚдҪңж–Үд»¶пјҢиҝҷжңүеҠ©дәҺжҸҗй«ҳжҖ§иғҪе’Ңе…је®№жҖ§гҖӮ

    1.2 жңүд»Җд№ҲдёҚеҗҢпјҹ

    WSL2 зҡ„жҠҖжңҜжҳҜеҹәдәҺHyper-vпјҢе№¶дё”Hyper-vжҳҜеҫ®иҪҜWindowsдё“дёҡзүҲ/ж——иҲ°зүҲиҮӘеёҰзҡ„

    ејҖеҗҜWSL2иҷҡжӢҹеҢ–жҠҖжңҜеҗҺпјҢзӣҙжҺҘеҸҳжҲҗиҷҡжӢҹжңәдәҶ

    1. дјҳзӮ№: LinuxеҸҜд»ҘзӣҙиҝһжҳҫеҚЎ,еҸҜд»ҘдҪҝз”ЁCUDA

    2. жңүдәҶиҝҷдёӘеҸҜд»ҘеңЁWindowsдёӢдҪҝз”ЁdockerдәҶ

    3. еҸҜд»ҘзӣҙжҺҘеңЁз»Ҳз«ҜеҗҜеҠЁиҷҡжӢҹжңәbashпјҢйқһеёёйқһеёёж–№дҫҝ

    4. еңЁLinuxдёӢдҪҝз”ЁwindowsиҪҜд»¶пјҢеңЁwindowsдёҠдә«еҸ—Linuxеә“пјҢжҜ”еҰӮж·ұеәҰеӯҰд№ зҡ„дёҖдәӣеә“

    Type-1 Hypervisor

    Type-1 Hypervisorжң¬иә«е°ұжҳҜдёҖдёӘеҹәжң¬зҡ„ж“ҚдҪңзі»з»ҹпјҢжҲ‘д»¬еңЁе…¶дёҠиҝҗиЎҢиҷҡжӢҹжңәгҖӮеӣ жӯӨпјҢиҝҗиЎҢжӯӨзұ»Hypervisorзҡ„зү©зҗҶжңәеҸӘиғҪз”ЁдәҺиҷҡжӢҹеҢ–зӣ®зҡ„пјҢдёҚиғҪз”ЁдәҺе…¶д»–зӣ®зҡ„гҖӮ

    дјҳзӮ№пјҡ

    • **жҖ§иғҪеҮәиүІпјҡ**дёҚеҸ—ж“ҚдҪңзі»з»ҹеӣәжңүйҷҗеҲ¶зҡ„зәҰжқҹпјҢж•…еҸҜд»ҘжҸҗдҫӣеҮәиүІзҡ„жҖ§иғҪгҖӮ
    • **й«ҳеәҰе®үе…Ёпјҡ**з”ұдәҺзӣҙжҺҘеңЁжІЎжңүд»»дҪ•еә•еұӮж“ҚдҪңзі»з»ҹзҡ„зү©зҗҶзЎ¬д»¶дёҠиҝҗиЎҢпјҢеӣ жӯӨдёҚдјҡеҸ—еҲ°ж“ҚдҪңзі»з»ҹйҖҡеёёзү№жңүзҡ„зјәйҷ·е’ҢжјҸжҙһзҡ„еҪұе“ҚгҖӮеҗҢж—¶еҸҜд»ҘжңҖеӨ§зЁӢеәҰдёҠзЎ®дҝқжҜҸдёӘиҷҡжӢҹжңәйғҪдёҺд»»дҪ•жҒ¶ж„ҸиҪҜд»¶жҙ»еҠЁйҡ”зҰ»ејҖжқҘгҖӮ

    Type-2 Hypervisor

    иҝҷзұ»Hypervisorжүҳз®ЎеңЁж“ҚдҪңзі»з»ҹдёҠпјҢе№¶дё”жҳҜдҪңдёәдёҖдёӘиҪҜд»¶иҝҗиЎҢеңЁзі»з»ҹеұӮдёҠд»ҘеҗҜз”ЁиҷҡжӢҹеҢ–гҖӮ

    дјҳеҠҝ

    • **з®ЎзҗҶз®ҖеҚ•пјҡ**жң¬иҙЁдёҠд№ҹжҳҜз®ЎзҗҶжҺ§еҲ¶еҸ°пјҢж— йңҖе®үиЈ…еҚ•зӢ¬зҡ„иҪҜд»¶еҢ…жқҘз®ЎзҗҶеңЁtype-2 hypervisorдёҠиҝҗиЎҢзҡ„иҷҡжӢҹжңәгҖӮ
    • **ж–№дҫҝжөӢиҜ•пјҡ**дҫҝдәҺжөӢиҜ•ж–°иҪҜд»¶жҲ–з ”з©¶йЎ№зӣ®пјҢеҸҜд»Ҙз®ҖеҚ•жҳ“иЎҢең°иҝҗиЎҢе…·жңүдёҚеҗҢж“ҚдҪңзі»з»ҹзҡ„еӨҡдёӘе®һдҫӢпјҢжқҘжөӢиҜ•иҪҜд»¶еңЁеҗ„дёӘзҺҜеўғдёӯзҡ„е·ҘдҪңж–№ејҸгҖӮ
